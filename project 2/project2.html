HTML (HyperText Markup Language) is the standard language used to create and design web pages. It forms the skeleton of all websites, defining the structure and layout of web content. HTML uses tags to create elements, with tags enclosed in angle brackets. Most tags come in pairs, such as an opening tag and a closing tag, like This is a paragraph.

An HTML document begins with the declaration, which specifies the document type and version of HTML. The entire document is enclosed within tags. The basic structure of an HTML document includes a head and a body. The head contains meta-information about the document, such as the title and links to external resources like CSS stylesheets and JavaScript files. The body contains the actual content of the webpage, including text, images, links, and other media.
